Examples to write cucumber test using page-object gem

## Requirements

- Chruby (https://github.com/postmodern/chruby)
- Bundler

## Getting Started

Clone the repository:

    git clone git://github.com/tancnle/page-object-examples.git

Run bundler:

    bundle install

Run cucumber features

    bundle exec rake

## Dependencies

- page-object
